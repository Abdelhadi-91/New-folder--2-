# CIGO Card — Showcase

Smart Payment System for Urban Transit · ESI Sidi-Bel-Abbès

## Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion

## Dev
```bash
npm install
npm run dev
```

## Public Assets
Place your assets in `public/`:
- `public/images/fcard.png` — Card front
- `public/images/bcard.png` — Card back
- `public/videos/1.mp4` — Validator demo
- `public/videos/2.mp4` — Recharge terminal demo
- `public/videos/3.mp4` — Mobile app demo
- `public/videos/4.mp4` — Admin dashboard demo
